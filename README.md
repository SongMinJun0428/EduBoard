# 🚀 EduBoard: 스마트 학급 관리 및 진로 탐색 플랫폼

EduBoard는 기존의 단조로운 학급 관리 도구를 넘어, **게이미피케이션(Gamification)**과 **인공지능(AI)**을 결합하여 학생들에게는 즐거움을, 관리자에게는 효율적인 통찰을 제공하는 통합 웹 플랫폼입니다.

---

## 🌟 주요 핵심 기능

### 1. 게이미피케이션 기반 학생 대시보드 (`index.html`)
- **실시간 성장 프로필**: 레벨, XP(경험치), 코인 시스템을 통해 학생의 활동을 시각화합니다.
- **다양한 미니게임 (10종+)**: 4는 10, 미로 탈출, 반응 속도 테스트 등 10가지 이상의 게임을 통해 코인을 획득할 수 있습니다.
- **상점 및 커스텀**: 획득한 코인으로 특별한 칭호와 이름을 꾸미는 색상 아이템을 구매할 수 있습니다.
- **학급 정보 공유**: 공지사항, 수업 자료실, 급식 및 시간표 정보를 실시간으로 확인합니다.

### 2. AI 기반 진로 밸런스 게임 (`job.html`)
- **가치관 기반 분석**: 단순한 선택지가 아닌, 선택의 '이유'를 AI(Google Gemini)가 분석하여 학생의 잠재적 진로 성향을 도출합니다.
- **높은 신뢰도**: 10만 번 이상의 시뮬레이션을 통해 검증된 균형 잡힌 알고리즘을 사용합니다.
- **상세 결과 리포트**: 9가지 직업 유형 아키타입과 그에 따른 맞춤형 조언을 제공합니다.

### 3. 강력한 관리자 콘솔 (`admin.html`)
- **실시간 데이터 분석**: 전체 사용자 수, 총 코인 유통량, 최고 레벨 등 학급의 주요 KPI를 한눈에 파악합니다.
- **정밀 필터링**: 학년, 반, 역할별로 사용자를 검색하고 개별 코인 지급 및 권한 관리가 가능합니다.
- **DB 익스플로러**: Supabase와 연동되어 데이터베이스 테이블을 직접 조회하고 관리할 수 있습니다.

## 🛡️ 보안 프로토콜 (Security)

본 프로젝트는 학생들이 개발자 도구(F12)를 통해 정보를 조작하는 것을 방지하기 위해 다음과 같은 3중 보안 체계를 갖추고 있습니다.

1.  **DOM 변조 방지 (MutationObserver)**: `security.js`를 통해 코인 점수나 스탯을 강제로 수정하려는 시도를 실시간으로 감지하고 즉시 원래 값으로 되돌립니다.
2.  **권한 기반 접근 제어 (RBAC)**: 관리자 페이지는 세션 확인뿐만 아니라 데이터베이스의 역할(Role) 정보를 실시간 교차 검증하여 비인가자의 접근을 원천 차단합니다.
3.  **상태 격리 (State Isolation)**: 주요 로직과 변수들을 즉시 실행 함수(IIFE) 내부로 격리하여 브라우저 콘솔에서 직접적인 변수 접근 및 수정을 어렵게 만들었습니다.

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (Layout), CSS3 Animations
- **Backend / DB**: Supabase (PostgreSQL, Realtime APIs, Auth System)
- **AI Engine**: Google Gemini API (via Supabase Edge Functions)
- **Library**: font-awesome (icons), Google Fonts (Inter, Pretendard)

## 📂 파일 구조

```text
/
├── admin.html          # 관리자 대시보드
├── index.html          # 학생 메인 페이지
├── job.html            # 진로 밸런스 게임 페이지
├── class.html          # 학급 전용 페이지
├── css/                # 스타일시트 모음
│   ├── index.css
│   ├── admin.css
│   └── job-*.css
└── js/                 # 자바스크립트 로직 모음
    ├── index.js        # 메인 앱 로직
    ├── admin.js        # 관리자 기능 전용
    ├── security.js     # 변조 방지 보안 스크립트
    ├── games.js        # 미니게임 통합 로직
    └── job-*.js        # 진로 분석 및 AI 연동
```

## 📝 관리자 가이드

1.  **로그인**: 관리자로 설정된 계정으로 로그인 시 상단 네비게이션에 '관리자 설정' 버튼이 활성화됩니다.
2.  **보안**: `security.js`가 활성화되어 있어 소스 코드 보기는 가능하나 실제 조작은 불가능합니다.
3.  **데이터 업데이트**: 모든 통계 수치는 Supabase 실시간 데이터를 기반으로 최신화됩니다.
